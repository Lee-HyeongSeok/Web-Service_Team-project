## 2차 설계 내용   
#### 주제 : 스터디 게시판 사이트 만들기   
***   

## 코어 기능   
	* 회원가입(사용자), 관리자 제외   
	* 관리자는 데이터베이스에 따로 개인 정보를 가진다.   
	* 로그인 및 로그아웃 기능   

## 사용자 기능   
	* 게시글 작성 / 수정 / 삭제   
	* 카테고리 선택 기능(select box)   
	* 브라우저 게시글 마다 댓글 폼 지원   
	* 사용자 자신이 올린 게시글 및 댓글 확인 기능   

## 관리자 기능   
	* 모든 게시글 삭제 권한   
	* 회원 수, 게시글 통계를 통한 가시적 사이트 유동인구 확인   
	* 카테고리 생성, 수정, 삭제   

***   

## 개발 환경   
	1. JSP - front, back   
	2. MySQL   
	3. Eclipse(IDE)
